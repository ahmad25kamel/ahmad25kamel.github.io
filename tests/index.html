Test Runner

Unit tests for Private Online Tools

📋 About Testing

These tests verify that the HTML-based tools function correctly in the browser. All tests run client-side and include:

  • Functional Tests: Core functionality of each tool
  • UI Tests: Interface elements and user interactions
  • Error Handling: Proper handling of edge cases
  • Browser Compatibility: Cross-browser functionality

Future Test Suites

Additional test suites can be added here for other tools:

  • 📄 PDF Tools Tests (Split, Merge, Convert)
  • 🖼️ Image Tools Tests (Resize, Compress)
  • 🔧 General UI/UX Tests
  • 🌐 Browser Compatibility Tests

🧪 How to Use

  1. Click on any test suite link to run the tests
  2. Tests will open in a new tab and run automatically
  3. Results will show passed/failed tests with details
  4. Check browser console for additional debugging information

🔧 Running Tests Locally

To run tests locally, serve this directory with a local web server:

python3 -m http.server 8000

Then visit: http://localhost:8000/tests/
